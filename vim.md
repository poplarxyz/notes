# vim
## cursor

## cut
- v
- d dd d^ d$
- x,y m z

## yank
- y 

## insert

- u Ctrl+r

## tab
- tab to spacae : set et && set et!
- replace exist sapace or tab: %retab!

## tips
- ZZ == :wq
- zz redraw the screen and center the cursor

## muti-file
- `ls` view file list
- `b number` enter file

## line break visable
- `set list`
- `nolist`
